## Comandos do Django:

### Criar um projeto:
```bash
django-admin startproject nome_do_projeto
```

### Criar um servidor:
```bash
python manage.py runserver
```

### Criar uma aplicação:
```bash
python manage.py startapp nome_da_app
```

### Sempre que algo for alterado no db:
```bash
python manage.py makemigrations
```
e o:
```bash
python manage.py migrate
```
